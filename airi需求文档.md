# AI 助手下位机系统需求文档

**硬件平台**: ESP32-P4
**更新日期**: 2026-04-02

---

## 1. 系统概述

本系统基于 ESP32-P4 开发板实现一个 USB 外设型 AI 助手下位机，通过 USB 连接 PC，承担音频输入/输出、显示等人机交互功能。PC 端运行 AI 处理（STT + LLM + TTS），ESP32-P4 负责实时音视频数据的高速传输与显示。

```
┌─────────────┐    USB 2.0 HS    ┌─────────────┐     Wi-Fi      ┌─────────────┐
│    PC       │◄═══════════════►│  ESP32-P4   │◄═════════════►│  AI 云服务   │
│  (AI 处理)   │   480 Mbps     │  (下位机)    │   局域网      │   (LLM)     │
│  RTX 4060   │  UAC + HID     │  800x480    │               │             │
└─────────────┘                └─────────────┘                └─────────────┘
```

## 2. 硬件配置

| 组件 | 型号/规格 | 说明 |
|------|----------|------|
| **主控** | ESP32-P4 | 双核 RISC-V @ 400MHz |
| **显示** | RGB LCD 800×480 | 24-bit RGB 接口 |
| **音频 Codec** | ES8388 | I2S 接口，支持播放+录音 |
| **USB** | USB 2.0 HS OTG | USB Audio Class Device |
| **存储** | SPI Flash | 程序存储 |
| **扩展** | XL9555 | GPIO 扩展芯片 |
| **网络** | Wi-Fi STA | 局域网连接 PC |

## 3. 功能需求

### 3.1 USB Audio Class (UAC) 设备

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **UAC 播放** | 接收 PC 下发的音频数据，通过 ES8388 播放 | P0 |
| **UAC 录音** | 采集麦克风数据，上传给 PC | P0 |
| **音量控制** | 支持静音、音量调节 | P1 |
| **采样率** | 支持 16kHz / 48kHz | P1 |

### 3.2 显示功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **LCD 驱动** | RGB LCD 800×480 @ 60Hz | P0 |
| **帧缓冲** | 双缓冲，支持 DMA 传输 | P1 |
| **显示旋转** | 支持横屏/竖屏切换 | P2 |

### 3.3 USB HID 设备

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **音量按键** | 音量+、音量-、静音快捷键 | P2 |
| **设备状态** | LED 指示灯（电源、网络、USB 连接状态） | P2 |

### 3.4 网络功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| **Wi-Fi 连接** | STA 模式连接局域网 | P1 |
| **状态上报** | 向 PC 推送设备状态 | P2 |

### 4.1 目录结构

```
main/
├── main.c              # 入口文件
├── APP/
│   ├── app_uac.c       # UAC 应用层
│   ├── app_lcd.c       # LCD 应用层
│   ├── app_hid.c       # HID 按键处理
│   └── app_vendor.c    # 厂商定制功能
├── UAC/
│   ├── usb_device_uac.c # UAC 设备驱动
│   └── uac_config.h    # UAC 配置

├── BSP/
│   ├── LCD/            # LCD 驱动 (rgblcd)
│   ├── ES8388/        # 音频Codec驱动
│   ├── MYI2S/         # I2S总线驱动
│   ├── LED/           # LED控制
│   └── XL9555/        # GPIO扩展芯片
└── Middlewares/
    └── tinyusb_src/   # TinyUSB协议栈
```
